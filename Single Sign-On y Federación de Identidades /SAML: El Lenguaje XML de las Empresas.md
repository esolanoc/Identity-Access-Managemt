## SAML - Security Assertion Markup Lenguage: 

Es el estándar abierto basado en XML utilizado para la autenticación SSO
 
##Assertion: 
Es el corazón de SAML , es básicamente un paquete de información en formato XML que contiene statements sobre el usuario autenticado.
 
### Incluye: 
NameID: Identificador del usuario ej. estebansolano@company.com
Attributes Statement: Atributos como el rol, title, departamento
Condition: Temporal de validez, audiencia permitida
