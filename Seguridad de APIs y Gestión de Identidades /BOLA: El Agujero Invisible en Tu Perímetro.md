##Broken Object Level Authorization- BOLA:
Es la vulnerabilidad número uno según OWASP API Security Top 10 de 2023. 40 % representa ataques a API's

El atacante autentica legítimamente con sus credenciales, obtiene un token válido, luego manipula el identificador en la URL. Si el sistema no verifica la propiedad del objeto, el atacante accede a datos ajenos.
