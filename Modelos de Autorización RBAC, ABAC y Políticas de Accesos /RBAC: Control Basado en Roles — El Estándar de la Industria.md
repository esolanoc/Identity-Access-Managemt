## 🔑 RBAC
Es un modelo de control de accesos basado en roles y es el modelo predominante para la administración avanzada de permisos
 
### 📜 Se rige por tres reglas:

1.	Asignación de roles: La persona debería ser agregada a un rol para poder asignarlo
2.	Autorización de roles: El role debe ser autorizado para ser usuario
3.	Autorización de permisos: El usuario puede ejecutar una acción siempre y cuando el permiso este autorizado para el rol activo
